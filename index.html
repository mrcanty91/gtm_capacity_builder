ASSUMPTION LEDGER

Defend Every Number

Nothing goes to the CFO as "we think". Every material number gets an owner and a paper trail; work the list until it's all AGREED, then export the brief.

// HOW IT WORKS
1. In Team Setup, Model Drivers or the Plan Builder, click the + chip beside any number to track it here.  2. Set the owner once — it acts as a tag (e.g. "VP Sales") that groups this page.  3. The group debates in the comment trail; status moves PROPOSED → CHALLENGED → AGREED. Board Review agents can push challenges here automatically.  4. When everything material reads AGREED, export the defendability brief for the CFO pack.
BOARD REVIEW

Face the Skeptics

Four operators interrogate the live model the way a real exec team would — assumptions challenged, timing questioned, attrition checked. Push any challenge straight into the ledger for the owning team to answer.
